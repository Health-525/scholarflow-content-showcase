# 图片目录说明

当前公开展示版优先使用脱敏后的结构图、示意图和安全截图，不直接放未经检查的真实界面截图。

## 推荐图片命名

- `content-overview.png`：内容层目录或阅读入口总览
- `input-entry-example.png`：`09-日常处理/` 输入入口示意
- `output-example.png`：课表、日报或周报代表性产物截图
- `dual-repo-architecture.png`：内容层与执行层协作图

## 使用建议

- README 中优先引用结构图或已脱敏截图
- 如果使用真实截图，必须先移除姓名、地点、时间线、头像、账号等可识别信息
- 如果暂时没有可公开图片，可先使用 Mermaid 图或静态示意图补位

## 当前状态

本目录当前只提供命名规范与使用边界，后续可以按上述文件名逐步补齐展示素材。
